## PRICING STRATEGY

### Tiered Pricing

| Plan | Price | Downloads | Features |
|------|-------|----------|----------|
| **Basic** | ₹199 | 1 PDF | Resume optimization, ATS keywords, PDF format |
| **Pro** | ₹499 | 3 PDFs | Basic + LinkedIn optimization, cover letter |
| **Premium** | ₹999 | 10 PDFs | Pro + Priority support, branding tips |

### Price Psychology

- **Anchor**: Show Premium first with ₹999
- **Decoy**: Make Pro look like best value
- **Entry**: ₹199 is impulse buy territory
- **Friction**: Remove friction from ₹199 tier

## CONVERSION OPTIMIZATION

### Funnel Stages

```
Visit → Upload → Select Plan → Payment → Download
  ↓         ↓          ↓         ↓        ↓
100%    30%       20%       60%      95%
```

### Conversion Bottlenecks

1. **Upload → Select Plan** (30% → 20%)
   - Problem: Users drop off at plan selection
   - Fix: Show value before asking for payment

2. **Select Plan → Payment** (20% → 60%)
   - Problem: Payment friction
   - Fix: UPI/QR code option, reduce steps

3. **Payment → Download** (60% → 95%)
   - Problem: Technical failures
   - Fix: Better error handling, email backup
